Firestore Rules Helper

If a page shows "Missing permissions", add this rule to your Firebase Console → Firestore → Rules:

match /hub_playlists/{d} {
  allow read, write: if request.auth != null;
}
match /hub_activity/{d} {
  allow read, write: if request.auth != null;
}
match /app_sessions/{d} {
  allow read, write: if true;
}
match /community_posts/{d} {
  allow read: if true;
  allow create: if true;
  allow update, delete: if request.auth != null;
}
